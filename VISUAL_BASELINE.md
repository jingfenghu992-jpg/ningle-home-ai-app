# VISUAL BASELINE
当前线上视觉基线（以实际运行版本为准）。

- **Theme**：暖色家居风（浅底 + 深绿强调色），移动端单列布局（max-width 约 375–430）。
- **核心色系**：
  - 背景：暖米色（`#F5F2ED`）
  - 卡片底：暖白（`#FFFCF7`）
  - 强调色：深绿（`#1F4D3A` / `#173C2D`）
  - 文本：暖深灰（`#4A453C` / `#2F2A23`）
- **Typography**：系统字体，较大行高（聊天文本易读）。

All future changes must maintain this visual identity.
