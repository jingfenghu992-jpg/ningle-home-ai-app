# 廚房與浴室價格 (Kitchen & Bath Pricing)

*所有櫃體均採用 ENF 級環保板材 / 防潮板。價格含設計、安裝及物流。*

## 1. 廚房地櫃 (Base Cabinet)
-   **高度**：3呎
-   **計價**：按直呎
-   **經濟實用款**：**HK$2,180 / 直呎** (防潮底板 + 標準導軌)
-   **高端配置款**：**HK$2,480 / 直呎** (防潮底板 + Blum導軌)

## 2. 廚房吊櫃 (Wall Cabinet)
-   **高度**：2.5呎
-   **計價**：按直呎
-   **經濟實用款**：**HK$1,880 / 直呎** (層板 + 門板)
-   **高端配置款**：**HK$2,180 / 直呎** (鋁框門 + LED燈)

## 3. 浴室櫃 (Bathroom Cabinet)
-   **高度**：2.5呎以內
-   **計價**：按件
-   **經濟實用款**：**HK$3,500 / 件** (ENF級防潮板 + 鏡櫃)
-   **高端配置款**：**HK$3,800 / 件** (ENF級防潮板 + 鋁框鏡門 + LED照明)
